ZenSheet

From   Lakebolt Research

Reactive Computing

  • ZenSheet resembles a spreadhseet tool, but is much more powerful
  • Computations are represented as a set of formulas and data inputs
  • ZenSheet reacts to changing inputs and automatically recalculates results
  • Inputs can be tied to data feeds, for continuous real-time processing
  • Creating and testing ZenSheet computation models is simple and intuitive
  • Anyone who knows how to use a spreasheet knows how to use ZenSheet!
